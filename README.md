### Included games:
- Even/Odd;
- Calculations;
- Greatest Common Divisor;
- Progression;
- Prime;

### Rules:
To win the game you should give three right answers in a row.

### Installation:
Clone the repo:
```
git clone
```
In root project folder:
```
make install
make publish
npx link
npx link @hexlet/code
```
After this commands quiz will be available from terminal by the names:

`brain-even`

`brain-calc`

`brain-gcd`

`brain-progression`

`brain-prime`
